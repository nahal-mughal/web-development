Git and Github

  1. Download git from https://git-scm.com/

  2. git init in terminal to initialize git

  3. git add . in terminal to add all files to git

  4. git commit -m "Added project files" in terminal to commit all changes to git

  5. git branch -M main in terminal to specify the branch

  6. Go to https://github.com and make a repository

  7. Copy the remote repository URL (https://github.com/yourusername/your-repo-name.git)

  8. git remote add origin https://github.com/yourusername/your-repo-name.git

  9. git push -u origin main to push your local commits to the remote repository

  10. Track the changes in your github repository
