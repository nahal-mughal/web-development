Git and Github

  1. Download git from https://git-scm.com/

  2. Enter command git init in terminal to initialize git

  3. Enter command git add filename.txt in terminal to add a file to git

  4. Enter command git add . in terminal to add all files to git

  5. Enter command git commit -m "Added project files" in terminal to commit all changes to git

  6. Go to https://github.com and make a repository

  7. Copy the remote repository URL (it looks like https://github.com/yourusername/your-repo-name.git)

  8. Enter command git add origin main https://github.com/yourusername/your-repo-name.git

  9. Enter commant git push -m origin main/master to push your local commits to the remote repository

  10. Track the changes in your github repository
